  ПАКЕТ ДЛЯ СБОРКИ FOLDER-HELPER.EXE НА WINDOWS 11
═══════════════════════════════════════════════════════════════

📦 СОСТАВ ПАКЕТА:
─────────────────
1. folder-helper.py          - Основной файл приложения
2. build-windows.bat          - Скрипт автоматической сборки
3. folder-helper-windows.spec - Конфигурация PyInstaller
4. WINDOWS_BUILD.md           - Подробная инструкция

🚀 БЫСТРАЯ СБОРКА:
─────────────────
1. Скопируйте все файлы на Windows-машину
2. Запустите: build-windows.bat
3. Готовый exe будет в папке: dist\folder-helper.exe

📋 ТРЕБОВАНИЯ:
──────────────
- Windows 10/11
- Python 3.8+ (скачать с python.org)
- Интернет для установки зависимостей (Flask, PyInstaller)

⚠️  ВАЖНО:
──────────
Готового exe-файла нет, так как PyInstaller не может собрать 
Windows exe на Linux. Сборку нужно выполнить на Windows-машине.

💡 АЛЬТЕРНАТИВЫ:
───────────────
1. Собрать на Windows-машине (домашний/рабочий компьютер)
2. Использовать виртуальную машину Windows
3. Попросить коллегу собрать для вас
4. Использовать облачный Windows (Azure, AWS)

📖 ПОДРОБНАЯ ИНСТРУКЦИЯ:
───────────────────────
См. файл: WINDOWS_BUILD.md
